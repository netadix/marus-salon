## 1. GitHubアカウント作成

### 手順

1. **GitHubにアクセス**
   - https://github.com/ を開く

2. **Sign up(登録)をクリック**
   - 右上の「Sign up」ボタンを押す

3. **必要情報を入力**
   - Username(ユーザー名): 半角英数字(例: marussalon)
   - Email address: メールアドレス
   - Password: パスワード(15文字以上推奨)

4. **メール認証**
   - 登録したメールアドレスに届いた認証コードを入力

5. **プランを選択**
   - **「Free」を選択**(無料プラン)

✅ **完了!** アカウントが作成されました

---

## 2. リポジトリ(保管庫)作成

### リポジトリとは？
ホームページのファイルを保管する場所です。

### 手順

1. **GitHubにログイン**
   - https://github.com/login

2. **新規リポジトリ作成**
   - 右上の「+」→「New repository」をクリック

3. **リポジトリ設定**
   
   **Repository name(必須):**
   ```
   marus-salon
   ```
   (※半角英数字とハイフンのみ使用可能)

   **Description(説明・任意):**
   ```
   マルズサロン公式サイト
   ```

   **Public / Private:**
   - **「Public」を選択**(公開)
   - ※Privateだと有料プランが必要

   **Initialize this repository with:**
   - ☑ **「Add a README file」にチェック**

4. **「Create repository」をクリック**

✅ **完了!** リポジトリが作成されました

---

## 3. ファイルのアップロード(コミット)

### 手順

1. **リポジトリページを開く**
   - https://github.com/あなたのユーザー名/marus-salon

2. **ファイルをアップロード**
   - 「Add file」→「Upload files」をクリック

3. **ファイルを選択**
   
   **アップロードするファイル:**
   - `index.html` (ホームページのファイル)
   
   **方法1: ドラッグ&ドロップ**
   - ファイルを画面にドラッグして放す
   
   **方法2: ファイル選択**
   - 「choose your files」をクリックしてファイルを選択

4. **コミットメッセージを入力**
   
   **Commit changes(変更内容):**
   ```
   初回アップロード
   ```
   (※何を変更したか分かるメモ)

5. **「Commit changes」をクリック**

✅ **完了!** ファイルがアップロードされました

---

## 4. GitHub Pages で公開

### 手順

1. **Settings(設定)を開く**
   - リポジトリページの「Settings」タブをクリック

2. **Pagesを開く**
   - 左メニューの「Pages」をクリック

3. **公開設定**
   
   **Source(ソース):**
   - 「Deploy from a branch」を選択
   
   **Branch(ブランチ):**
   - 「main」を選択
   - 「/ (root)」を選択
   
   **「Save」をクリック**

4. **公開URLを確認**
   
   数分待つと、画面上部に以下が表示されます:
   ```
   Your site is live at https://あなたのユーザー名.github.io/marus-salon/
   ```

✅ **完了!** ホームページが公開されました!

## 5. 内容の更新方法

### HTMLファイルの編集方法

#### 方法A: GitHub上で直接編集

1. **ファイルを開く**
   - リポジトリページで「index.html」をクリック

2. **編集モードに入る**
   - 右上の鉛筆アイコン(Edit)をクリック

3. **内容を編集**
   - テキストを変更
   - 例: 営業時間、メニュー、価格など

4. **変更を保存**
   - 「Commit changes」をクリック
   - コミットメッセージを入力(例: 「営業時間を更新」)
   - 「Commit changes」をクリック

5. **反映を確認**
   - 数分待ってサイトをリロード

#### 方法B: パソコンで編集してアップロード

1. **HTMLファイルをダウンロード**
   - 「index.html」をクリック
   - 「Raw」→右クリック→「名前を付けて保存」

2. **テキストエディタで開く**
   - メモ帳(Windows)またはテキストエディット(Mac)
   - または、Visual Studio Code(推奨)

3. **内容を編集**
   - 変更したい部分を探して編集

4. **保存**
   - 上書き保存

5. **GitHubに再アップロード**
   - 「Add file」→「Upload files」
   - ☑ 「Commit directly to the main branch」
   - 「Commit changes」

✅ **完了!** 内容が更新されました

## 6. よくある質問

### Q1. 公開URLを短くしたい

**現在のURL:**
```
https://あなたのユーザー名.github.io/marus-salon/
```

**短くする方法:**

リポジトリ名を「あなたのユーザー名.github.io」にすると:
```
https://あなたのユーザー名.github.io/
```
になります。

**注意:** これは1アカウント1つまでです。

---

### Q2. 独自ドメイン(marussalon.com)を使いたい

1. **ドメインを取得**(年間1,000円〜)
   - お名前.com、ムームードメインなど

2. **GitHub Pagesで設定**
   - Settings → Pages → Custom domain に入力

3. **DNS設定**
   - ドメイン会社の管理画面でDNS設定

※詳しい手順は別途ご相談ください(有償対応)

---

### Q3. ファイルが反映されない

**原因と対処法:**

1. **キャッシュの問題**
   - ブラウザの「スーパーリロード」
   - Windows: `Ctrl + F5`
   - Mac: `Cmd + Shift + R`

2. **ファイル名の間違い**
   - 必ず「index.html」(小文字)

3. **公開設定の確認**
   - Settings → Pages で「main」ブランチが選択されているか

---

### Q4. HTMLファイルの編集箇所が分からない

**よく変更する箇所:**

#### 営業時間を変更
```html
<div class="info-value">
  10:00〜16:00<br>     ← ここを変更
  （最終受付14:45）    ← ここを変更
</div>
```

#### 料金を変更
```html
<div class="menu-price">¥8,000〜</div>  ← ここを変更
```

#### 住所を変更
```html
奈良県橿原市四条町627-5<br>  ← ここを変更
```

#### 電話番号を追加
```html
<div class="info-item">
  <div class="info-label">Tel</div>
  <div class="info-value">000-0000-0000</div>  ← 追加
</div>
```
